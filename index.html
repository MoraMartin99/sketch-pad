Sketch pad

  • Lápiz (left-click para alternar con el cursor)

    Mueva el cursor por el canvas para colorear. Por defecto el color cambiará aleatoriamente por cada celda que pinte, en configuraciones puede fijar un color: Configuraciones > Color > Personalizado.

    Borrador (spacebar para alternar con el cursor)

    Ubique el cursor en una celda para borrar su color. Para borrar todo el canvas: Configuraciones > Canvas > Limpiar.
